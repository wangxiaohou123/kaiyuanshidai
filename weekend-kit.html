🗣️ Section A · 周末话术手册（C05 接触剧本 V2）
核心话术升级说明：两会确认长护险成"第六险"后，开元的切入话术从"我们在做养老服务"升级为"帮您家长者免费申领800元/月政府补贴"。这个话头没有商业目的感，反而是在做好事，居委会/街坊邻居接受度极高。
场景 A-1 · 优先级 P0
跟家人（居委会任职）说的话
目标：请家人帮忙介绍辖区内有失能长者的家庭
▼
使用时机：周末跟家人吃饭/闲聊时自然带出，不要郑重其事地"开会"，越随意越好。
开场（切勿直接说"我要创业"）
"妈/爸/[称谓]，你们居委会那边，有没有家里有老人卧床、或者需要人照顾的情况？我最近在研究一个政策，国家现在有个补贴，失能的老人每个月能领800块钱的政府消费券，专门用来请人上门照护的。我在想，你们那边有没有什么家庭知道这个事？"
→ 目的：探问有无失能长者家庭，同时传递"800元补贴"话题，不提任何商业动作
如果对方有线索（有认识的家庭）
"那我能不能让你帮我转达一下，就说有个渠道可以帮他们申领这个补贴，流程比较复杂，但我们可以帮他们弄。不用花他们什么钱，就是搞清楚他们家的情况，然后帮他们递材料。你看能不能帮我问问哪家有需要？"
→ 目的：争取拿到1-2个可以联系的家庭信息，不谈钱不谈服务合同
如果对方问"你为什么关心这个"
"我在研究养老这个方向，两会刚出来的政策，我觉得挺有意思的。而且你们那边肯定有很多老人家不知道这个补贴，挺可惜的。我就想试试能不能帮几个家庭先用上这个政策，也顺便了解一下大家的需求。"
→ 关键：定位为"公益性了解"，不是"找客户"
如果对方态度积极，可以进一步问
"那这样，你帮我问一下，他们家老人大概是什么情况？是完全卧床、还是生活能自理但需要帮忙？家里有没有子女照顾？子女上班吗？我就是先了解了解情况，后面看看怎么帮他们。"
→ 收集：失能程度 + 是否有照护者 + 子女工作情况（付费能力信号）
本场景成功标准：对方表示"我帮你问问"或者直接给你介绍了1个家庭的基本情况。不需要当场见面，不需要确认任何合作意向。
场景 A-2 · 机会性触达
直接遇到街坊邻居时的话
适用于：在广外小区遇到有长者的家庭，随机切入
▼
识别信号：推着老人轮椅散步的子女 / 抱怨"照顾老人太累"的街坊 / 楼道里见到明显失能的长者家庭。
轻松切入（遇到推轮椅/陪老人的）
"阿姨/叔叔，您家老人是在家照顾吗？现在国家有个新政策，失能的老人每个月政府给800块钱的消费券，可以请专业人上门帮忙，您知道吗？"
→ 不等对方反应，先把"800元/月"这个利益点抛出去
如果对方感兴趣，继续
"这个钱是政府发的，不是商业的东西。就是失能老人可以申领，然后用这个券请人上门帮忙洗澡、翻身、做护理。现在申领流程比较麻烦，一般家庭自己不太会弄。
我在帮几个家庭试点申领，您有没有兴趣了解一下？就是登记一下情况，看老人是否符合条件，不收您任何费用。"
→ 关键词："政府发的""不收费""帮您申领"——降低警惕心
留联系方式话术
"这样，您留个微信/电话，我发您一份说明，里面有申领条件和流程。您家老人符合的话，我帮您跑一趟街道，申请材料我帮您准备。"
→ 成功收集联系方式 = 本次接触成功，E06第一行填进去
场景 A-3 · 后续跟进
有初步意向后的上门沟通话术
适用于：家人帮介绍了家庭，需要当面了解情况时
▼
上门开场（由家人引荐的情况）
"您好，我叫王晶晶，是[家人称谓]介绍过来的。我最近在帮几个家庭研究怎么申领政府的养老补贴。听说您家有老人需要照顾，我来了解一下情况，看看符不符合申领条件，完全不收费的。"
了解情况（按顺序问，自然聊天方式）
1. "老人多大了？平时生活能自理吗，还是需要人照顾？"
2. "有没有评过失能等级？或者做过医疗评估？"（没有就说可以帮他们申请评估）
3. "现在谁在照顾？您自己吗，还是另外请了人？"
4. "一个月大概花多少在照护上？"（了解付费能力和现状）
5. "您最希望解决什么问题？是您太累了、还是找不到合适的人、还是钱不够用？"
→ 第5个问题是最重要的，答案直接决定我们服务的切入点
结束时的承诺（不过度许诺）
"好，我把您家的情况记一下，我去查一查申领条件是否符合，大概一两周内给您回复。如果符合，我帮您把材料整理好，一起去街道办理，全程我陪着您。"
→ 不说"我们有服务"，只说"帮您申领补贴"——这是储备期的定位
